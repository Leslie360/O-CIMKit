# 🌀 Chaotic Lorenz 混沌轨迹器件存算一体时序预测

本目录包含了基于易失性短时记忆器件（Volatile）的储层计算（Reservoir Computing, RC）三维混沌洛伦兹吸引子（Lorenz Attractor）轨迹时序预测系统。混沌系统的特点是“蝴蝶效应”——对初始条件极度敏感且长时不可预测。本系统展示了如何利用有机突触晶体管的物理电导衰减特性作为自然时序积分器，在写噪声约束下高保真地重建混沌时序。

---

## ⚡ 2. 物理器件映射与仿真特性

系统绑定易失性晶体管模型（对应配置文件 `profiles/repository/PFO_AlOx_Volatile.json`）：
1. **短时物理记忆 (Short-term Volatile Memory)**：
   器件在连续输入信号刺激下，电导会发生暂时性上升，并在无外部刺激时依照物理常数 $\tau_{\text{volatile}} = 3.64\text{s}$ 进行指数衰减（弛豫效应）。这种特性天生替代了数字端的时序延迟单元。
2. **状态写入随机涨落**：
   每一次状态转变中，包含高斯分布的物理电导涨落噪声，系统利用读出层正则化对其进行平抑。

---

## 🧠 3. 计算模型与算法架构

1. **Lorenz 动力学信号生成**：
   使用四阶 Runge-Kutta 算法数值求解 Lorenz 经典三维非线性微分方程组，生成空间轨迹序列。
2. **单通道储层演化 (Reservoir State Evolution)**：
   Lorenz 序列经缩放后作为电流/照度信号注入物理储层，利用器件的电导弛豫动态产生高维时序状态映射。
3. **线性读出层 (Readout Regressor)**：
   提取物理电导的时序响应矩阵，通过带有 L2 惩罚项的岭回归（Ridge Regression）求解得到输出权重矩阵：
   $$Y(t) = W_{out} \cdot X_{\text{state}}(t)$$
   在数字端或忆阻器阵列中完成下一时间步三维坐标的快速预测。

---

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **时间常数物理阻抗对齐 (Timescale Tuning)**：
  混沌预测的关键是储层器件的物理消散时间 $\tau$ 必须与混沌系统演化步长 $\Delta t$ 在同一量级上。我们通过阻抗对齐，将易失器件的衰减弛豫速度与 Lorenz 采样率强对齐，使储备池表现出最佳的“回声状态属性（Echo State Property）”。
* **读出层正则化平抑 (Ridge Regularization)**：
  器件电导散布会导致 $X_{\text{state}}(t)$ 矩阵存在多重共线性和随机偏置。我们在岭回归损失函数中引入了最优的 L2 惩罚项，极大地提升了模型在应对物理高斯噪声干扰时的系统级健壮性。
