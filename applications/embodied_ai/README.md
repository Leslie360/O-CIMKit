# 🤖 Embodied AI 具身智能多模态触觉材料存算融合分类

本目录包含了基于易失性/非易失性混合突触器件的具身智能多模态触觉材料分类系统。在仿生机器人执行抓取任务时，通过手指表面的压力与摩擦力滑移时间序列信号识别物体材质（glass, sandpaper, rubber, metal），对精细操作至关重要。本系统展示了在物理噪声约束下，如何通过时间常数对齐和量化感知训练实现几乎无损的极高触觉感知分类准确率。

## ⚡ 2. 物理器件映射与仿真特性

系统绑定易失性/非易失性协同器件模型（对应配置文件 `profiles/repository/PFO_AlOx_Volatile.json` 与 `PFO_AlOx_NonVolatile.json`）：
1. **短时易失弛豫 (Volatile Dynamic)**：
   物理储层器件弛豫常数可编程控制，通过模拟电导衰减实现时序滑移动态特征滤波。
2. **64 态非易失读出量化 (64-state NV Quantization)**：
   读出层全连接权重被映射为分立的 64 电导阶梯，并在反向传播中使用直通估计器（STE）进行微调以吸收器件非线性偏差。

---

## 🧠 3. 计算模型与算法架构

1. **时序触觉滑移信号生成**：
   生成包含摩擦系数时变性、滑动速度波动以及接触压力正弦扰动的时序滑移数据。
2. **时间常数物理对齐 (Timescale Alignment)**：
   触觉信号通过易失性储层（RC）进行滤波与特征投影，其衰减常数 $\tau_{\text{volatile}}$ 进行宽频段多项扫描。
3. **QAT 分类器**：
   提取物理储层状态的均值、标准差、最大/最小值等统计特征，送入 4 层量化感知 MLP 神经网络分类器：
   $$\text{Input (8)} \rightarrow \text{QATMLP (128)} \rightarrow \text{QATMLP (64)} \rightarrow \text{QATMLP (32)} \rightarrow \text{Output (4)}$$

---

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **自适应时间常数扫描 (Timescale Sweeping)**：
  触觉信号的特征周期与摩擦表面的粗糙度及抓取滑动速度密切相关。我们通过在大范围内自动扫描器件物理时间常数 $\tau \in [0.1\text{s}, 30.0\text{s}]$，寻找分类准确率的极值，成功将器件的电导消散周期对齐到材料滑移特征频率（最佳 $\tau$ 为 3.64s，分类率达 **99.0%**）。
* **QAT 权重剪裁与 STE 反向传导**：
  将前向计算中的全连接权重映射到 64 阶梯中，并限制极值，反向通过 STE 规避电导台阶导数为零的物理障碍，使读出器能自适应吸收忆阻器的 C2C 波动。
