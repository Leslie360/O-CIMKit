# 🗣️ RAVDESS 语音情感多尺度自注意力储层存算量化诊断

本目录包含了基于多尺度自注意力物理储层（Multi-Scale Attention Reservoir）特征提取与非易失忆阻器量化感知训练（QAT）的语音情感识别系统。语音情绪信号在时域上波动剧烈且包含多种尺度特征。本系统展示了在存在忆阻器多级电导离散与高斯写入噪声的真实约束下，如何通过软硬件协同技术达到学术前沿水平。

## ⚡ 2. 物理器件映射与仿真特性

系统集成了易失性器件（用于储层演化）与非易失忆阻器模型（对应配置文件 `profiles/repository/PFO_AlOx_NonVolatile.json`，用于读出分类）：
1. **短时弛豫物理自滤波 (Volatile Dynamic)**：
   物理储层晶体管利用电导自发衰减性质，对输入的语音 MFCC（梅尔频率倒谱系数）序列在物理端实现特征增强和非线性维度拓展。
2. **64 态非均匀电导量子台阶 (64-state Readout)**：
   读出全连接层权重被映射为分立的 64 态电导阻值对，在前向传播中产生舍入噪声。
3. **电导写入 Cycle-to-Cycle (C2C) 高斯涨落**：
   突触电导在写入脉冲时会有漂移。本系统利用量化感知训练（QAT）来适应这种偏差。

## 🧠 3. 计算模型与算法架构

1. **时域语音声学输入**：
   语音文件首先在数字端提取 39 维梅尔频率倒谱系数（13 维 MFCC + 13 维一阶差分 + 13 维二阶差分），输入物理储层。
2. **多尺度自注意力储备池 (Multihead Attention Reservoir)**：
   - 输入向量乘以权重送入 400 个神经元的储层中，在 5 种不同衰减常数（$\tau \in [2.0, 5.0, 10.0, 20.0, 40.0]$）下进行状态演化。
   - 对演化后的状态历史进行物理多头自注意力融合（Multihead Self-Attention, 4个头）：
     $$\text{Output} = \text{Softmax}\left(\frac{Q K^T}{\sqrt{d_k}}\right) V$$
     融合多尺度的时序长依赖关联，输出提取特征。
3. **QAT MLP 读出分类器**：
   $$\text{Input (fused features)} \rightarrow \text{QATMLP (256)} \rightarrow \text{QATMLP (4)}$$
   网络前向使用 64 阶梯进行电导量化，反向使用直通估计器（STE）更新。

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **物理端多头自注意力机制 (Attention Fusion)**：
  这是提升语音情感识别率的核心所在。语音语调的起伏发生在不同的时间尺度（如音节与整句）。通过 5 种不同的衰减常数对齐各频段，并引入 4 头自注意力对时序历史进行物理级加权融合，成功将语音的宏观语境与微观音色特征完美捕获，弥补了浅层 MLP 拟合长时依赖的劣势。
* **STE 直通估计与物理截断**：
  前向台阶截断，反向恒等求导，防止梯度在 64 态下消散；将全连接权重严格限制在物理区间内，规避饱和误差。
* **余弦学习率衰减与 Dropout 干扰对抗**：
  降低模型在 QAT 细微电导变动中的扰动敏感度。
