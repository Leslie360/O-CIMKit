# ⚙️ CWRU 工业轴承故障忆阻器存算一体量化分类诊断

本目录包含了基于非易失性 64 电导状态忆阻器（Memristor）量化感知训练（QAT）的 CWRU 旋转机械轴承故障诊断系统。在高噪声的工业生产环境中，轴承故障的早期微弱冲击特征容易被淹没。本系统展示了如何通过硬件感知协同设计（HAT），在硬件写噪声及多级量化的约束下实现几乎零精度损失的工业级故障分类。

## ⚡ 2. 物理器件映射与仿真特性

系统绑定忆阻器模型（对应配置文件 `profiles/repository/PFO_AlOx_NonVolatile.json`）。该器件高保真仿真了以下**物理非理想约束**：
1. **64 态分立电导量子台阶**：
   电导物理值被离散限制为 64 个能级阶梯，造成参数量化误差。
2. **直通不可微导数梯度**：
   突触电导在台阶过渡时梯度为零，采用 LSQ QAT 中融合的直通估计器（STE）解决。
3. **Cycle-to-Cycle (C2C) 写入电导散布**：
   模拟电导在连续擦写脉冲下的高斯漂移噪声。

## 🧠 3. 计算模型与算法架构

1. **时域振动信号输入**：
   读取 CWRU 凯斯西储大学轴承驱动端/风扇端的加速度振动信号（12kHz 采样频率），进行时域切片预处理。
2. **量化感知多层感知机 (QAT MLP)**：
   设计了 4 层神经网络：
   $$\text{Input (1000)} \rightarrow \text{QATMLP (128)} \rightarrow \text{QATMLP (64)} \rightarrow \text{QATMLP (32)} \rightarrow \text{Output (4)}$$
   全连接层权重使用 QATMLPLayer 进行差分映射，在前向传播中直接套用 64 态离散电导规则，在反向传播中应用 STE 更新梯度。

---

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **直通估计器 (Straight-Through Estimator, STE)**：
  前向使用离散台阶量化，反向传播时强行令阶跃函数导数为 1，确保了参数梯度在离散空间内能稳定传导。
* **电导极性截断 (Weight Range Clipping)**：
  严格限制网络权重绝对值在 1.0 以内，完美对齐物理器件的最大/最小电导限制（$G_{\text{max}}/G_{\text{min}}$），规避了超出物理电导量程导致的梯度消散。
* **余弦退火学习率**：
  使网络权重在精调期稳定收敛至量化电导低阻区。

## 🚀 5. 快速启动与参数说明

### 运行 CWRU 轴承诊断训练与评估：
```bash
python3 main.py bearing_fault --epochs 50 --batch-size 32 --lr 0.001
```

### 关键命令行参数：
* `--epochs`：QAT 训练轮数（推荐 50 轮）。
* `--batch-size`：推荐设为 32。
* `--lr`：初始学习率（默认 0.001）。
* `--device-profile`：指定器件配置，默认载入 `profiles/repository/PFO_AlOx_NonVolatile.json`。
