# 🫵 NIST 指纹忆阻器存算一体分类识别

本目录包含了基于 28 电导状态忆阻器（Memristor）差分电导映射的指纹分类诊断系统。指纹识别要求捕捉细腻的边缘、分叉和环形拓扑特征。本系统展示了在忆阻器非理想写散布与离散化状态限制下，如何通过硬件感知协同设计（HAT）保障在真实场景下的极高识别准确率。

---

## ⚡ 2. 物理器件映射与仿真特性

系统绑定忆阻器器件模型（对应配置文件 `profiles/repository/FingerMemristor.json`）。在指纹识别处理中，本框架高保真地仿真了以下**物理非理想效应**：
1. **28 态非均匀电导离散**：
   电导物理值只能落在 28 个分立的状态区间内。
2. **Cycle-to-Cycle (C2C) 和 Device-to-Device (D2D) 写入涨落**：
   突触电导在写入脉冲时有随机漂移，噪声幅度服从高斯分布，会在模型推理时引入扰动。
3. **电导状态演化非理想 (LTP/LTD Non-ideality)**：
   包含增益 LTP 与衰减 LTD 过程中的非线性斜率变化。

## 🧠 3. 计算模型与算法架构

1. **高阶特征提取器 (CNN/ResNet)**：
   使用 ResNet 主干提取指纹图像的宏观及微观纹路特征。
2. **存算差分电导映射**：
   通过电导差分映射：
   $$W \propto G_{\text{pos}} - G_{\text{neg}}$$
   在线或离线将最终读出权重部署于 28 状态忆阻器阵列。

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **Mixup 数据混合增强**：
  为防止网络在非理想器件物理权重的干扰下发生过拟合，我们引入了 Mixup 增强算法：
  $$\tilde{x} = \lambda x_i + (1 - \lambda) x_j, \quad \tilde{y} = \lambda y_i + (1 - \lambda) y_j$$
  这强迫模型在中间决策区域产生平滑过渡，从而极大地平抑了由于器件电导 C2C 波动带来的突发误差。
* **推理期动态批量归一化校准 (BatchNorm Calibration)**：
  在评估前对 BN 层的 `running_mean` 和 `running_var` 进行无梯度的小批量更新，将图像的批统计分布与器件物理噪声偏置拉齐。
* **余弦学习率衰减 (Cosine Annealing)**：
  采用平滑退火策略使梯度下降平稳收敛。

## 🚀 5. 快速启动与参数说明

### 运行指纹分类训练与评估：
```bash
python3 main.py fingerprint_rec --epochs 120 --batch-size 32 --lr 0.002
```

### 关键命令行参数：
* `--epochs`：训练轮数（推荐 120 - 150 轮）。
* `--batch-size`：推荐设为 32。
* `--lr`：初始学习率（默认 0.002）。
* `--device-profile`：指定器件配置，默认载入 `profiles/repository/FingerMemristor.json`。
