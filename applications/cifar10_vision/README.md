# 👁️ CIFAR-10 仿生光电存算一体图像分类

本目录包含了基于有机电化学晶体管（OECT）光电突触器件的 CIFAR-10 仿生光电影像识别系统实现。通过结合有机光敏材料的响应物理特性与神经网络，本系统展示了在存在强光电噪声与电阻更新非理想性的恶劣物理环境下，依然能保持高准确率图像分类的硬件感知协同设计（HAT）。

## ⚡ 2. 物理器件映射与仿真特性

系统采用 OECT 绿色光电突触器件模型（对应配置文件 `profiles/repository/OECT_Vision.json`）。在图像处理过程中，每个像素的照度值被物理映射为器件接收到的光子数量。在此过程中，本框架高保真地仿真了以下**物理非理想效应**：
1. **光电泊松噪声 (Poisson Shot Noise)**：
   在光捕获阶段，入射光强 $I$ 转换为光子数 $N$ 服从泊松分布 $N \sim \text{Poisson}(\lambda)$，其中 $\lambda \propto I \times \text{max\_photons}$。在低照度下，泊松噪声带来的信噪比恶化会导致图像分类崩溃。
2. **光电响应与电导饱和**：
   光照激发的激子积累与电导更新表现出非线性，且电导有物理最大值 $G_{\text{max}}$ 和最小值 $G_{\text{min}}$ 限制。
3. **电导写入非理想性 (LTP/LTD Non-ideality)**：
   模拟电导在连续写入脉冲下的非对称上升/下降曲线，包含 Cycle-to-Cycle (C2C) 和 Device-to-Device (D2D) 的电导随机散射。

## 🧠 3. 计算模型与算法架构

1. **光电变换与逆 ISP 映射**：
   输入图像先通过仿生视网膜模型，模拟光电突触的光激活过程。
2. **特征提取主干**：
   采用轻量化高效的 ResNet 结构（支持参数化定义通道数与深度），对光电映射后的噪声信号提取高阶特征。
3. **差分权重映射**：
   网络权重被分解为 $W \propto G_{\text{pos}} - G_{\text{neg}}$ 的正负电导对，并在电导离线写入或在线 QAT 中应用非线性 LTP/LTD 曲线约束。

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

为了在物理噪声下锁定 **90%+** 的高准确率，本应用集成了以下极致优化手段：
* **光敏感度自适应校准 (Photon Flux Parameterization)**：
  在强光照（例如晴天，`max_photons = 10000.0`）下，泊松噪声比例降至极低，使信噪比大幅提升。
* **双通道重新累计 BatchNorm 校准 (BatchNorm Calibration)**：
  因为物理器件的写入噪声和非线性变换，预训练模型提取的特征会发生严重的协变量平移。在训练完成后，调用校准函数，使用训练集的小批量样本不更新梯度、仅重新累计 BN 层的 `running_mean` 和 `running_var`，即可完美消除分布平移，恢复坍塌的准确率。
* **带有权重衰减的 AdamW 优化**：
  在多项式非线性拟合电导时，配合余弦学习率衰减（Cosine Annealing），使网络权重收敛到电导平缓区域。

## 🚀 5. 快速启动与参数说明

### 运行完整训练与评估：
```bash
python3 main.py cifar10_vision --epochs 40 --batch-size 128 --max-photons 10000.0
```

### 关键命令行参数：
* `--epochs`：训练的总轮数（推荐 40 轮以达到最优收敛）。
* `--batch-size`：批次大小（默认 128）。
* `--lr`：初始学习率（默认 0.001）。
* `--max-photons`：最大光子数通量（默认 10000.0 代表高照度晴天，可降低至 20.0 模拟极暗光夜视环境）。
* `--device-profile`：指定器件配置，默认载入 `profiles/repository/OECT_Vision.json`。
