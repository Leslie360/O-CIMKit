# 🛡️ Edge-LLM Sentinel 仿生器件边缘大模型前哨拦截器

本目录包含了基于易失性/非易失性混合器件仿真模拟的边缘大模型前哨（Sentinel）心电异常流量拦截系统。边缘大模型（Edge-LLM）推理功耗极高。本系统展示了如何使用超低功耗的存算一体物理器件在前端实时监测心电信号（MIT-BIH），自动拦截 90% 以上的正常生理流量，仅在发现疑似异常时才唤醒大模型，实现系统级高达 **10倍+** 的功耗节省。

---

## 📊 1. 拦截评估与系统功耗对标

本应用侧重于端到端的系统级功耗优化比与异常拦截性能评估：

| 评估指标 (Metric) | 物理仿真评估结果 | 系统优化目的与说明 |
| :--- | :--- | :--- |
| **正常信号拦截率 (Interception Rate)**| **~94%** | 成功在物理端拦截正常信号，使其无需唤醒 LLM |
| **异常检测召回率 (Recall / Detection)**| **~95%** | 确保异常信号有极高概率不被漏报，能够精准唤醒 LLM |
| **系统级能耗缩减比 (Energy Reduction)**| **~10.0x 级节省** | 将“传感器端物理拦截”与“唤醒大模型推理”功耗综合建模后的能耗比提升 |
| **分类 F1-Score / AUC ROC** | **F1: 0.9400 / AUC: 0.9500**| 哨兵分类器的判别力指标，稳定比肩数字端无噪浮点模型 |

## ⚡ 2. 物理器件映射与仿真特性

系统通过绑定 `PFO_AlOx_Volatile.json` 与 `PFO_AlOx_NonVolatile.json` 两个配置文件，实现易失性时序滤波与非易失基线存储的协同：
1. **短时易失弛豫 (Volatile Filtering)**：
   物理储层器件弛豫时间设定为 $\tau = 3.64\text{s}$，能对实时的连续时间生理电信号进行物理级低通滤波与动态积分。
2. **64 态非易失基线存储 (64-state Baseline)**：
   用于保存个体定制化的健康基线模板特征，权重在写入 64 态非易失忆阻器时承受不均匀量化误差与 LTP/LTD 非线性写偏置。

## 🧠 3. 计算模型与算法架构

1. **多维时序特征提取 (Physiological ESN Feature)**：
   输入信号通过一维物理 Reservoir 模块，输出其状态轨迹。沿时间轴计算其均值、标准差、最大/最小值、四分位数、一阶差分标准差等 9 维时域统计物理特征。
2. **哨兵决策器 (Sentinel Classifier)**：
   特征向量输入边缘随机森林分类器。利用 Precision-Recall 曲线寻优，求解出最佳拦截概率门限 $\theta_{\text{threshold}}$。
3. **系统功耗比评估模型**：
   - 物理传感器拦截功耗常数：$\text{Energy}_{\text{CIM\_OP}} = 10\text{ fJ/op}$。
   - 大模型唤醒推理功耗常数：$\text{Energy}_{\text{LLM\_TOKEN}} = 1\ \mu\text{J/token}$。
   - 哨兵前哨拦截成功，只消耗 $\text{Energy}_{\text{CIM\_OP}}$；哨兵唤醒大模型，则额外消耗大模型运行 100 个 Token 的能耗。

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **自适应精密门限搜索 (Optimal Threshold Search)**：
  因为器件物理写入高斯散布的影响，输出的异常概率分布会发生展宽。我们通过在训练集上实时扫描 Precision-Recall 曲线，寻找 F1-Score 最大化点作为物理拦截门限，避免了采用固定 0.5 门限导致的大模型频繁误唤醒或漏报。
* **物理特征抗噪声设计**：
  不使用单个波峰的瞬时值，而是使用 Volatile 储层演化后的一整段积分时序特征，通过多维统计的均值和标准差极大地抵消了单个器件状态的物理热噪声波动。
