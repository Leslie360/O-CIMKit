# 🧠 Sleep-EDF 脑电疲劳分期多尺度储层存算量化诊断

本目录包含了基于多尺度物理储层（Multi-Scale Reservoir）时序特征提取与 3 层非易失忆阻器 QAT 读出器的脑电（EEG）疲劳分期/睡眠 staging 诊断系统。脑电信号具有微弱、多频段耦合以及复杂的长时序依赖特征。本系统展示了在 64 电导状态器件物理噪声干扰下，通过时间常数对齐和网络加深如何将前馈存算一体诊断的精度推到极致。

---

## 📊 1. 学术 SOTA 与平台精度对比

在 Sleep-EDF 生理信号（3分类：Awake, Light Sleep, Deep Sleep）诊断任务中，器件物理约束下的精度如下：

| 运行模式 (Mode) | 精度/评估指标 (Metric) | 物理环境与噪声配置 | 核心优化技术及说明 |
| :--- | :--- | :--- | :--- |
| **Digital Baseline** (纯算法无噪) | **75.60%** | 无物理噪声，32位浮点精度 | 多尺度 RC + 3层 MLP 读出器 |
| **Device Simulation** (本平台物理仿真) | **71.95%** (调优中可达 **73% - 75%**) | 64态非易失忆阻器 + 写入噪声 + 电导非线性 | **多尺度时间常数对齐 (3.0s, 10s, 30s) + 3层QAT分类器加深 + Dropout (0.4) 抑制** |
| **Academic SOTA** (学术最先进水平) | **84.00% - 87.00%** | 纯数字端复杂算法 | 依赖 Bi-LSTM/Transformer 等**长时序网络**（而目前 CIM 忆阻器阵列对前馈网络支持最佳，故存在物理架构上限） |

## ⚡ 2. 物理器件映射与仿真特性

系统集成了易失性器件（用于储层演化）与非易失忆阻器模型（对应配置文件 `profiles/repository/PFO_AlOx_NonVolatile.json`，用于读出分类）：
1. **易失物理弛豫 (Volatile Dynamic)**：
   利用易失器件的短时记忆电导衰减，对输入的脑电时序振荡信号在物理端完成积分和非线性变换。
2. **64 态非均匀电导离散 (64-state Readout)**：
   读出分类器参数限制在分立的 64 个电导台阶中，在前向传播中产生严重的精度舍入噪声。
3. **电导写入高斯散布**：
   电导写入时产生的高斯漂移会破坏特征层均值，通过 QAT 协同 STE 机制进行抑制。

## 🧠 3. 计算模型与算法架构

1. **多尺度物理储层 (Multi-Scale Reservoir, RC)**：
   将双通道脑电信号分别输入 3 个具有不同衰减时间常数（$\tau = 3.0\text{s}, 10.0\text{s}, 30.0\text{s}$）的 Reservoir 物理层中。脑电的多频段特征（$\alpha, \beta, \theta, \delta$ 波）在不同尺度下被物理捕捉，合并输出 9000 维的高维特征。
2. **3层 QATMLP 分类器**：
   $$\text{Input (9000)} \rightarrow \text{QATMLP (512)} \rightarrow \text{QATMLP (256)} \rightarrow \text{QATMLP (3)}$$
   网络全连接层权重使用 QATMLPLayer，通过 STE 对 64 个离散电导状态进行在线梯度估计，输出 3 分类。

---

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **物理多时间尺度融合 (Multi-Scale Fusion)**：
  脑电信号中包含快速的梭形波与慢速的深睡慢波。设计多时间常数（3s, 10s, 30s）并联 ESN，成功在物理端对齐了多段脑电生理时间尺度，避免了在数字端进行大量 STFT / MFCC 特征变换的功耗开销。
* **读出分类网络加深与宽度扩增 (Hyperparameter Scaling)**：
  相较于原本的简单 2 层 MLP，我们将读出器升级为 3 层 QAT 架构，并将隐藏层维度拓宽至 512 维，极大地增强了在有限的 64 电导状态约束下的非线性空间划分能力。
* **高 Dropout 抗过拟合保护**：
  引入 `nn.Dropout(0.4)` 与权重正则，强迫网络学习不依赖单个忆阻器器件状态的鲁棒组合特征。
