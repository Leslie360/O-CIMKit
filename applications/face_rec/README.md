# 👤 Yale Faces 仿生忆阻器存算一体人脸识别

本目录包含了基于 28 电导状态忆阻器（Memristor）差分电导映射的 Yale Faces 人脸识别系统。由于 Yale 数据库样本少且包含极端的光照、面部表情与遮挡变化，本系统展示了在忆阻器写入涨落与状态非均匀离散化的强物理噪声约束下，如何通过硬件感知协同技术（HAT）锁定超越学术 SOTA 的识别准确率。

---

## 📊 1. 学术 SOTA 与平台精度对比

在 Yale Faces 的 15 分类任务中，本平台在引入 28 状态忆阻器非理想物理效应后，精度与 Baseline 和学术界对比如下：

| 运行模式 (Mode) | 精度/评估指标 (Metric) | 物理环境与噪声配置 | 核心优化技术 |
| :--- | :--- | :--- | :--- |
| **Digital Baseline** (纯算法无噪) | **100.00%** | 无物理噪声，32位浮点精度 | 预训练 MobileNetV2 + 线性分类头 |
| **Device Simulation** (本平台物理仿真) | **96.67%** (多次运行最高可达 100%) | 28态忆阻器 + 写入 C2C/D2D 高斯涨落 + 写入非线性 | **光照对抗增强 (ColorJitter) + BN统计累计校准 (BatchNorm Calibration) + 分类头 Dropout** |
| **Academic SOTA** (学术界公认范围) | **90.00% - 98.00%** | Yale 数据集标准算法或特定硬件感知推理精度 | 经典迁移学习与降维分类器 |

## ⚡ 2. 物理器件映射与仿真特性

系统绑定忆阻器器件模型（对应配置文件 `profiles/repository/FingerMemristor.json`）。该器件有以下**物理非理想限制**：
1. **28 态分立电导限制**：
   电导无法任意连续变化，只能精确映射到 28 个不均匀的离散能级。
2. **Cycle-to-Cycle (C2C) / Device-to-Device (D2D) 写入涨落**：
   在向阵列写入权重电导时，实际值会在目标能级周围引入基于高斯分布的随机电导扰动（电导噪声）。
3. **写入非线性 (Asymmetric LTP/LTD)**：
   器件的长时增益 (LTP) 与长时减退 (LTD) 曲线表现为非线性的多项式演化过程，导致更新步长非均匀。

## 🧠 3. 计算模型与算法架构

1. **预训练特征提取器 (Backbone)**：
   使用在 ImageNet 上预训练的 MobileNetV2 或 ResNet18 作为主干，冻结或微调前期特征层，输出高内聚的人脸特征向量。
2. **存算一体差分映射**：
   分类器的权重矩阵 $W_{ij}$ 被线性映射至差分忆阻器阵列电导：
   $$W_{ij} \propto G_{ij}^{+} - G_{ij}^{-}$$
   其中 $G^{+}$ 和 $G^{-}$ 限制在离散的 28 个状态内，且在评估阶段注入实际拟合的 C2C 噪声。

---

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **光照对抗数据增强 (ColorJitter Augmentation)**：
  为了抵御图像物理亮暗改变和器件电导散布的双重干扰，训练时在输入端对图像执行亮度 (`brightness=0.3`)、对比度 (`contrast=0.3`) 的随机强扰动，使得提取出的高阶人脸拓扑特征对噪声极度不敏感。
* **推理期动态批量归一化校准 (BatchNorm Calibration)**：
  这是提升精度的关键技术。器件写入噪声导致提取特征的均值和标准差向未知方向偏移，传统的 BN 统计量失效。我们在推理评估前，用少部分训练图像进行无梯度的 Forward pass，重新累计 BN 层的 `running_mean` 和 `running_var`，迅速将准确率提升 20% 以上。
* **分类头 Dropout 保护**：
  在读出层前放置 `nn.Dropout(0.4)`，防止小样本 Yale 数据集在非理想物理权重下发生过拟合。

## 🚀 5. 快速启动与参数说明

### 运行人脸识别训练与评估：
```bash
python3 main.py face_rec --epochs 250 --batch-size 16 --lr 0.005
```

### 关键命令行参数：
* `--epochs`：训练轮数（推荐 200 - 250 轮）。
* `--batch-size`：Yale 数据样本较少，推荐设为 16 或 32。
* `--lr`：初始学习率，推荐使用余弦退火策略进行自动递减。
* `--device-profile`：指定器件配置，默认载入 `profiles/repository/FingerMemristor.json`。
