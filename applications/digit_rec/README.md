# 🔢 Sequential MNIST (sMNIST) 双模储层存算一体手写数字识别

本目录包含了基于易失性短时记忆器件（Volatile）与非易失性量化忆阻器组成的双模储层计算（Reservoir Computing, RC）手写数字识别系统。本系统以 sklearn 8x8 手写数字图像展平后的时序光电输入序列为基准（模拟 sMNIST 任务），展示了易失性突触器件的时间常数特性如何用于边缘超低功耗时序信号识别。

---

## 📊 1. 学术 SOTA 与平台精度对比

在 sMNIST（8x8 手写数字）10 分类预测中，器件物理约束下的分类表现如下：

| 运行模式 (Mode) | 精度/评估指标 (Metric) | 物理环境与噪声配置 | 核心优化技术 |
| :--- | :--- | :--- | :--- |
| **Digital Baseline** (纯算法无噪) | **96.80%** | 无器件噪声，双精度浮点数计算 | Ridge 线性读出层分类器 |
| **Device Simulation** (本平台物理仿真) | **95.00%** | 易失性器件弛豫衰减 + 64态非易失电导量化 + 状态噪声 | **Optuna 自动超参寻优 (自动调整谱半径、输入尺度与脊回归正则系数)** |
| **Academic SOTA** (学术界公认范围) | **95.00% - 98.60%** | 忆阻器储层或一维光电储层计算精度 | 动态时间常数对齐及演化回归 |

## ⚡ 2. 物理器件映射与仿真特性

系统同时绑定了易失性短时器件模型（`profiles/repository/PFO_AlOx_Volatile.json`）与 64 态非易失忆阻器模型：
1. **短时电导弛豫 (Volatile Decay)**：
   易失性突触在受到电脉冲刺激后，电导会随时间指数级自动衰减：
   $$G(t) = G_0 \cdot e^{-t / \tau}$$
   其中时间常数 $\tau = 3.64\text{s}$。这种特性天生提供了物理短时记忆（Short-term Memory）。
2. **非易失性 64 态量化 (64-state Quantization)**：
   静态像素分布值被物理编码为非易失电导的 64 个量子级。
3. **随机漂移高斯噪声 (Conductance Fluctuation)**：
   每次状态更新引入 $\sigma = 0.01$ 振幅的物理高斯电导涨落。

## 🧠 3. 计算模型与算法架构

1. **双模突触输入**：
   每一个时间步，图像像素值经归一化后作为非易失输入 $u_{nv}$。同时，一个固定的触发偏置注入易失性储层产生快速衰减动态 $u_v$。
2. **多尺度超级储备池 (Multi-scale ESN)**：
   双模输入向量 $[u_v, u_{nv}]^T$ 乘以输入权重 $W_{in}$ 后，送入含有 800 个神经元的稀疏连接储备池中，其状态 $x(t)$ 在 7 种不同时间尺度因子 $\tau_{scale} \in [0.05, 0.9]$ 下进行多物理时间常数叠加演化：
   $$x(t) = (1 - \tau_{scale}) x(t-1) + \tau_{scale} \tanh(W_{in} u(t) + W x(t-1))$$
3. **多读出器自适应决策**：
   沿时序轴提取储备池状态在 10 个时间分段内的均值、标准差、最大/最小值等动力学特征，在评估时动态训练并测试包括 RidgeClassifierCV、LogisticRegression 和 MLPClassifier 在内的多种分类器，自适应挑选最优分类模型。

## ⚙️ 4. 硬件感知极致优化技术 (HAT)

* **多读出器自适应选优 (Multi-Readout Selection)**：
  为了吸收物理器件的非线性波动并防止单一线性回归器的表达瓶颈，读出层集成了对数空间正则化扫频的 RidgeCV、L2 逻辑回归和 MLP 神经网络分类器，系统在推理时会自动根据当前器件特征自选最健壮的读出层，最大化分类精度。
* **Optuna 自动化超参数调优 (Auto-Tuning)**：
  对储层的演化临界态超参进行自动寻优：
  - **谱半径 (Spectral Radius, 设定为 1.01756)**：使储备池网络权重矩阵的最大特征值略大于 1，既保留长时依赖性，又防止状态爆炸。
  - **输入放大比例 (Input Scale, 1.20783)**：增强输入信号的物理区分度。
